# Braga English Club — Website

Landing page for [business-english.club](https://business-english.club) —
British English lessons with a native speaker in Braga, Portugal.
Available online worldwide or in person.

## About the project

- **Instructor:** Toby Raynsford — multilingual language coach, Bangor University graduate
- **Lessons:** Business English, everyday English, 1:1 and paired sessions
- **Location:** Braga, Portugal (in person) + worldwide (online)

## Tech stack

- [Bootstrap](https://getbootstrap.com/) via [Start Bootstrap Creative](https://startbootstrap.com/theme/creative/) theme
- [Pug](https://pugjs.org/) — HTML templating
- [SCSS](https://sass-lang.com/) — styling
- JavaScript

## Local development

```bash
npm install
npm start        # live preview in browser with hot reload
```

## Build & deploy

```bash
npm run build    # compiles Pug, SCSS, JS into /dist
```

The `/dist` folder is published automatically via **GitHub Pages**
on every push to `master`.
